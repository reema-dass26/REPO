#	Title	Purpose	Query
1	Who ran the experiment?	Identify the agent (user/session)	sparql\nSELECT ?agentName WHERE {\n ?agent a prov:Agent ;\n <http://xmlns.com/foaf/0.1/name> ?agentName .\n}\n
2	Which dataset was used in a run?	Trace prov:used dataset in a training run	sparql\nSELECT ?dataset WHERE {\n ?run a prov:Activity ;\n prov:used ?dataset .\n FILTER CONTAINS(STR(?dataset), "dataset")\n}\n
3	Which model was generated by which run?	Trace prov:wasGeneratedBy linkage	sparql\nSELECT ?model ?run WHERE {\n ?model a prov:Entity ;\n prov:wasGeneratedBy ?run .\n}\n
4	Which code version was used in the training?	Identify source code (Git commit)	sparql\nSELECT ?codeHash WHERE {\n ?run a prov:Activity ;\n prov:used ?code .\n ?code <http://example.org/commit_hash> ?codeHash .\n}\n
5	When did the training start and end?	Training window for reproducibility	sparql\nSELECT ?start ?end WHERE {\n ?run a prov:Activity ;\n prov:startedAtTime ?start ;\n prov:endedAtTime ?end .\n}\n
6	List all training metrics	View output performance metrics	sparql\nSELECT ?metric ?value WHERE {\n ?run a prov:Activity ;\n ?metric ?value .\n FILTER CONTAINS(STR(?metric), "accuracy")\n}\n
7	Get model hyperparameters	View parameters for reproducibility	sparql\nSELECT ?param ?value WHERE {\n ?model a prov:Entity ;\n ?param ?value .\n FILTER CONTAINS(STR(?param), "hyper")\n}\n
8	Which model used a specific dataset?	Trace from dataset to output model	sparql\nSELECT ?model WHERE {\n ?run a prov:Activity ;\n prov:used <http://example.org/dataset/Iris> .\n ?model prov:wasGeneratedBy ?run .\n}\n
9	What preprocessing was applied?	Inspect data preparation	sparql\nSELECT ?key ?val WHERE {\n ?model <http://example.org/preprocessing_info> ?info .\n}\n
10	List all provenance types	Debug the graph itself	sparql\nSELECT DISTINCT ?type WHERE {\n ?s a ?type .\n}\n
